Sintaxe

Um documento CSS consiste em seletores que descrevem quais elementos e como eles serão modificados. Começamos descrvendo qual elementos queremos modificar no seletor, depois adicionamos atributos e o seu valor

        
            Seletor {
            atributo: valor;
            }

Tipos de Seletores

Estilo diretamente no alemento do HTML

        <p>Olá mundo</p>

        p{
        color:#f05a20;
        }

Podemos usar um ID no elemento. Interessante usar em elementos que não vão se repetir.

        <p id="sobre">Olá mundo</p>

        #sobre{
        color:#f05a20;
        }

Podemos usar um classe no elemento. Interessante usar em elementos que vão se repetir na página.

        <p class="sobre">Olá mundo</p>

        .sobre{
        color:#f05a20;
        }
